
# USO DE PROXY PC & ANDROID


## 1. PC
### 1.1 Windows
1. Ir a Configuración > Red e Internet > Proxy.
2. Activar "Usar un servidor proxy".
3. Dirección: `TU_IP_PUBLICA`
4. Puerto: `8443`

### 1.2 Linux
**Entorno grafico:**
1. Ir a Configuración del sistema > Red > Proxy.
2. Seleccionar "Manual" y rellenar los campos con IP y puerto.

**Terminal:**
1. Configura `HTTTP/HTTPS` (activo mientras durante la sesión del terminal)
```bash
export http_proxy="http://TU_IP_PUBLICA:8443" 
export https_proxy="http://TU_IP_PUBLICA:8443"
export no_proxy="localhost,127.0.0.1,::1" ##Evita que el localhost pase por el proxy
```
2. Si quieres que la configuración persista agrega la configuración anterior a tu .bashrc

```bash
nano ~/.bashrc
# Y aplica los cambios
source ~/.bashrc
```

## 2. Android
### 2.1 Desde Wifi

1. Conectate a una red Wi-Fi.
2. Ir a Ajustes > Red e Internet > Wi-Fi > Red actual > Avanzado.
3. En "Proxy", seleccionar "Manual".
4. Configurar:
   - **Nombre del host del proxy:** `TU_IP_PUBLICA`
   - **Puerto del proxy:** `8443`
5. Guardar y reconectarse a la red.
6. Probar desde navegador: https://check.torproject.org
